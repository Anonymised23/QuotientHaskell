INSTALLATION INSTRUCTIONS FOR QUOTIENT HASKELL
==============================================

1. Ensure that you have installed the z3 SMT solver (this is required to compile code with Liquid Haskell)
   - You can use brew with Mac, though Rosetta is necessary for M1/M2 architectures
   - Otherwise, just follow the instructions at https://github.com/Z3Prover/z3

2. Follow the Liquid Haskell build instructions found in the README of Quotient Haskell (found at https://github.com/Anonymised23/QuotientHaskell)
